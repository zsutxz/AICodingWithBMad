## 游戏概述
开发一款基于Unity的2D五子棋对战游戏，支持本地双人对战模式（无需联网与AI对手）。本项目将作为AI COding whit Bmad Method 开发Unity 2D游戏开发的实践案例。

## 开发技术栈
| 组件 | 技术选择 | 说明 |
|------|----------|------|
| 核心引擎 | Unity 2022.3 LTS | 稳定版本，长期支持 |
| 编程语言 | C# | 游戏逻辑实现 |
| UI系统 | Unity UI + TextMeshPro | 高质量文本渲染 |
| 音效管理 | Unity AudioSource | 基础音效播放 |
